# 每日时政新闻收集

自动收集国内外时政新闻，AI翻译外文标题，生成3-5个要点摘要，通过飞书消息推送。

## 触发条件

当用户表达以下意图时调用：
- "收集今天的新闻"
- "获取时政新闻"
- "运行新闻收集"
- "daily-political-news"

## 数据源

### 国内媒体
- **澎湃新闻** - https://www.thepaper.cn/
- **财新网** - https://www.caixin.com/

### 国际媒体
- **BBC News** - https://www.bbc.com/news
- **Reuters** - https://www.reuters.com/
- **Associated Press** - https://apnews.com/

## 新闻范围

- 时政
- 经济
- 社会
- 军事

## 输出格式

飞书消息格式：

```
📰 每日新闻简报 - YYYY-MM-DD

🇨🇳 国内新闻
━━━━━━━━━━━━━━━
1. 【新闻标题】
   来源：澎湃新闻
   链接：[原文链接]
   
   📋 要点：
   • 要点一
   • 要点二
   • 要点三

2. 【新闻标题】...

🇺🇸 国际新闻
━━━━━━━━━━━━━━━
1. 【英文标题 / 中文翻译】
   来源：Reuters
   链接：[原文链接]
   
   📋 要点：
   • 要点一
   • 要点二
   • 要点三

2. 【新闻标题】...
```

## AI处理

1. **外文翻译**：将英文标题翻译为中文
2. **要点提取**：从正文提取3-5个关键要点
3. **内容过滤**：筛选时政、经济、社会、军事相关新闻

## 定时配置

默认每天早上8点执行（周末照常）：

```bash
# OpenClaw Cron 配置
0 8 * * * /path/to/scripts/collect_news.py
```

## 使用方式

### 手动执行
```bash
python scripts/collect_news.py
```

### 通过 Agent 调用
```
用户：收集今天的新闻
→ 执行新闻收集流程
→ 推送飞书消息
```

## 技术实现

1. **RSS抓取**：从各媒体的RSS源获取最新新闻
2. **内容解析**：提取标题、链接、正文摘要
3. **AI处理**：调用LLM进行翻译和要点提取
4. **消息推送**：通过飞书API发送格式化消息

## 注意事项

1. **网络访问**：需要能访问各媒体网站
2. **RSS稳定性**：部分媒体RSS可能变动，需定期维护
3. **内容合规**：仅抓取公开新闻，不涉及付费内容
4. **频率控制**：避免频繁请求导致IP被封
