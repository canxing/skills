# Bug Fixing Skill

一个用于指导Claude按照严格流程修复bug的技能。

## 使用方式

在Claude Code中使用以下命令触发：

```
/skill-creator 修复bug
```

或者直接描述你的bug：

```
修复bug: [bug描述]
```

## 流程说明

本skill定义了7步bug修复流程：

1. **分析代码** - 分析bug原因并撰写说明，等待用户确认
2. **设计测试方案** - 设计能重现bug的测试用例方案，等待用户确认
3. **确保现有UT通过** - 运行现有单元测试，确保基线正常
4. **添加失败UT** - 编写会失败的单元测试来重现bug
5. **修复代码** - 最小化修改代码来修复bug
6. **使新增UT通过** - 验证新增测试通过
7. **使所有UT通过** - 确保没有引入回归问题

## 特点

- **测试驱动**: 先写测试后修复，确保bug被真正解决
- **用户确认**: 关键节点需要用户确认，避免误解
- **最小修改**: 只修改必要的代码，降低风险
- **回归保护**: 通过测试确保修复不会破坏现有功能

## 注意事项

- 每完成一步都会等待用户确认
- 可以随时回退到上一步
- 严格遵循"先失败测试后修复"的原则
